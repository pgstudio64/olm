3. Emprises totales par bloc × standard (desk + zones)

Emprise totale = dimensions desk + zones de chaque côté (non_superposable + candidate).
Les zones varient selon le standard et le type de bloc. Pour les blocs face-à-face (E/W) : zone = ES-01 + ES-06 (chair + passage).
Pour les blocs seul/côte (W seulement) : zone = ES-01 + (ES-03 − ES-01) = ES-03.

4. Règle d'affichage des distances (V-01)

Une distance inter-blocs est affichée si et seulement si :
1. L'un est le plus proche voisin de l'autre dans une direction cardinale (droite ou bas).
2. Les emprises desk (physiques, sans zones) des deux blocs se chevauchent sur l'axe perpendiculaire : chevauchement EO pour des voisins verticaux, chevauchement NS pour des voisins horizontaux.

Les zones de circulation/recul ne participent pas au test de chevauchement.
